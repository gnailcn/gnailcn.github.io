AcWing 106. 动态中位数

2023-10-29 进阶指南 AcWing 堆 算法

依次读入一个整数序列，每当已经读入的整数个数为奇数时，输出已读入的整数构成的序列的中位数。

输入格式

第一行输入一个整数 PPP，代表后面数据集的个数，接下来若干行输入各个数据集。

每个数据集的第一行首先输入一个代表数据集的编号的整数。

然后输入一个整数 MMM，代表数据集中包含数据的个数，MMM 一定为奇数，数据之间用空格隔开。

数据集的剩余行由数据集的数据构成，每行包含 101010 个数据，最后一行数据量可能少于 101010 个，数据之间用空格隔开。

输出格式

对于每个数据集，第一行输出两个整数，分别代表数据集的编号以及输出中位数的个数（应为数据个数加一的二分之一），数据之间用空格隔开。

数据集的剩余行由输出的中位数构成，每行包含 101010 个数据，最后一行数据量可能少于 101010 个，数据之间用空格隔开。

输出中不应该存在空行。

数据范围

1≤P≤10001 \le P \le 10001≤P≤1000,
1≤M≤999991 \le M \le 999991≤M≤99999,
所有 MMM 相加之和不超过 5×1055 \times 10^55×105。

输入样例：

3 
1 9 
1 2 3 4 5 6 7 8 9 
2 9 
9 8 7 6 5 4 3 2 1 
3 23 
23 41 13 22 -3 24 -31 -11 -8 -7 
3 5 103 211 -311 -45 -67 -73 -81 -99 
-33 24 56

输出样例：

1 5
1 2 3 4 5
2 5
9 8 7 6 5
3 12
23 23 22 22 13 3 5 5 3 -3 
-7 -3

程序

import java.util.Scanner;
import java.util.PriorityQueue;

public class Main
{
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
        for(int tt = 0; tt < T; tt ++)
        {
            int id = sc.nextInt();
            int n = sc.nextInt();
            PriorityQueue<Integer> bigPQ = new PriorityQueue<Integer>((x,y)->y-x);
            PriorityQueue<Integer> smallPQ = new PriorityQueue<Integer>();
            System.out.printf("%d %d\n", id, (n+1)/2);
            for(int i = 1; i <= n; i ++)
            {
                int x = sc.nextInt();
                insert(x, bigPQ, smallPQ);
                if(i%2==1)
                {
                    //奇数的时候中位数刚好是bigPQ.peek()
                    System.out.printf("%d ", bigPQ.peek());
                }
                if(i%20==0)
                {
                    System.out.println(); //有20保证后面有21，因为总数是奇数
                }
            }
            System.out.println();
        }
    }
    
    //需要保证 bigPQ.size() >= smallPQ.size() 并且最多相差1
    public static void insert(int x, PriorityQueue<Integer> bigPQ, PriorityQueue<Integer> smallPQ)
    {
        if(bigPQ.size()==0)
        {
            bigPQ.add(x);
            return;
        }
        int tpMid = bigPQ.peek();
        if(x <= tpMid)
        {
            //插入到左边的大根堆
            bigPQ.add(x);
            if(bigPQ.size()-smallPQ.size()>=2)
            {
                int top = bigPQ.remove();
                smallPQ.add(top);
            }
        }
        else
        {
            //插入到右边的小根堆
            smallPQ.add(x);
            if(smallPQ.size()>bigPQ.size())
            {
                int top = smallPQ.remove();
                bigPQ.add(top);
            }
        }
        return;
    }
}
