AcWing 110. 防晒

2023-11-17 贪心 进阶指南 AcWing 算法

有 CCC 头奶牛进行日光浴，第 iii 头奶牛需要 minSPF[i]minSPF[i]minSPF[i] 到 maxSPF[i]maxSPF[i]maxSPF[i] 单位强度之间的阳光。

每头奶牛在日光浴前必须涂防晒霜，防晒霜有 LLL 种，涂上第 iii 种之后，身体接收到的阳光强度就会稳定为 SPF[i]SPF[i]SPF[i]，第 iii 种防晒霜有 cover[i]cover[i]cover[i] 瓶。

求最多可以满足多少头奶牛进行日光浴。

输入格式

第一行输入整数 CCC 和 LLL。

接下来的 CCC 行，按次序每行输入一头牛的 minSPFminSPFminSPF 和 maxSPFmaxSPFmaxSPF 值，即第 iii 行输入 minSPF[i]minSPF[i]minSPF[i] 和 maxSPF[i]maxSPF[i]maxSPF[i]。

再接下来的 LLL 行，按次序每行输入一种防晒霜的 SPFSPFSPF 和 covercovercover 值，即第 iii 行输入 SPF[i]SPF[i]SPF[i] 和 cover[i]cover[i]cover[i]。

每行的数据之间用空格隔开。

输出格式

输出一个整数，代表最多可以满足奶牛日光浴的奶牛数目。

数据范围

1≤C,L≤25001 \le C,L \le 25001≤C,L≤2500,
1≤minSPF≤maxSPF≤10001 \le minSPF \le maxSPF \le 10001≤minSPF≤maxSPF≤1000,
1≤SPF≤10001 \le SPF \le 10001≤SPF≤1000

输入样例：

3 2
3 10
2 5
1 5
6 2
4 1

输出样例：

2

程序

import java.util.Scanner;
import java.util.Arrays;

public class Main
{
    //实现Comparable接口，重写compareTo方法
    public static class Cow implements Comparable<Cow>
    {
        int minv, maxv;
        public Cow(int minv, int maxv)
        {
            this.minv = minv;
            this.maxv = maxv;
        }
        //值为负数，表示当前对象小于参数对象，排在前面
        public int compareTo(Cow c)
        {
            return this.maxv - c.maxv;
        }
    }
    static final int N = 2505;
    static final int M = 1005;
    static Cow[] cows = new Cow[N];
    static int[] spf = new int[M];
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int m = sc.nextInt();
        for(int i = 0; i < n; i ++)
        {
            int minv = sc.nextInt();
            int maxv = sc.nextInt();
            cows[i] = new Cow(minv, maxv);
        }
        for(int i = 0; i < m; i ++)
        {
            int a = sc.nextInt();
            int b = sc.nextInt();
            spf[a] += b;
        }
        //按照右区间从大到小排序
        Arrays.sort(cows, 0, n);
        int ans = 0;
        //选择能选的强度最低（最左侧）的防晒霜
        //因为大的防晒霜要留给后面的牛
        for(int i = 0; i < n; i ++)
        {
            int minv = cows[i].minv;
            int maxv = cows[i].maxv;
            for(int j = minv; j <= maxv; j ++)
            {
                if(spf[j] > 0)
                {
                    spf[j] --;
                    ans ++;
                    break;
                }
            }
        }
        System.out.println(ans);
    }
}
