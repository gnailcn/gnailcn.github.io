AcWing 145.超市

2024-03-12 贪心 进阶指南 AcWing 堆 算法

原题

超市里有 N 件商品，每件商品都有利润 pi 和过期时间 di，每天只能卖一件商品，过期商品不能再卖。

求合理安排每天卖的商品的情况下，可以得到的最大收益是多少。

输入格式

输入包含多组测试用例。

每组测试用例，以输入整数 N 开始，接下来输入 N 对 pi 和 di，分别代表第 i 件商品的利润和过期时间。

在输入中，数据之间可以自由穿插任意个空格或空行，输入至文件结尾时终止输入，保证数据正确。

输出格式

对于每组产品，输出一个该组的最大收益值。

每个结果占一行。

数据范围

0≤N≤10000,
1≤pi,di≤10000
最多有 14 组测试样例

输入样例：

4  50 2  10 1   20 2   30 1

7  20 1   2 1   10 3  100 2   8 2
   5 20  50 10

输出样例：

80
185

程序

import java.util.Scanner;
import java.util.Arrays;
import java.util.PriorityQueue;

public class Main
{
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext())
        {
            int n = sc.nextInt();
            Goods[] goods = new Goods[n];
            for(int i = 0; i < n; i++)
            {
                int p = sc.nextInt();
                int d = sc.nextInt();
                goods[i] = new Goods(p, d);
            }
            Arrays.sort(goods);
            //小根堆
            PriorityQueue<Integer> pq = new PriorityQueue<Integer>();
            for(int i = 0; i < n; i++)
            {
                //将当前元素加入优先队列
                pq.add(goods[i].p);
                //如果堆里面元素大于当前天数, 则弹出堆顶元素
                if(pq.size() > goods[i].date)
                {
                    //弹出堆顶元素
                    pq.poll();
                }
            }
            //堆里面元素的和就是最大收益
            int sum = 0;
            while(!pq.isEmpty())
            {
                sum += pq.poll();
            }
            System.out.println(sum);
        }
    }

    public static class Goods implements Comparable<Goods>
    {
        int p;
        int date;
        public Goods(int p, int date)
        {
            this.p = p;
            this.date = date;
        }
        //排序规则: 日期早的排在前面
        public int compareTo(Goods g)
        {
            return this.date - g.date;
        }
    }
}
