AcWing 91. 最短Hamilton路径

2023-10-15 进阶指南 AcWing 动态规划 位运算 算法

给定一张 nnn 个点的带权无向图，点从 0∼n−10 \sim n-10∼n−1 标号，求起点 000 到终点 n−1n-1n−1 的最短 Hamilton 路径。

Hamilton 路径的定义是从 000 到 n−1n-1n−1 不重不漏地经过每个点恰好一次。

输入格式

第一行输入整数 nnn。

接下来 nnn 行每行 nnn 个整数，其中第 iii 行第 jjj 个整数表示点 iii 到 jjj 的距离（记为 a[i,j]a[i,j]a[i,j]）。

对于任意的 x,y,zx,y,zx,y,z，数据保证 a[x,x]=0，a[x,y]=a[y,x]a[x,x]=0，a[x,y]=a[y,x]a[x,x]=0，a[x,y]=a[y,x] 并且 a[x,y]+a[y,z]≥a[x,z]a[x,y]+a[y,z] \ge a[x,z]a[x,y]+a[y,z]≥a[x,z]。

输出格式

输出一个整数，表示最短 Hamilton 路径的长度。

数据范围

1≤n≤201 \le n \le 201≤n≤20
0≤a[i,j]≤1070 \le a[i,j] \le 10^70≤a[i,j]≤107

输入样例：

5
0 2 4 5 1
2 0 6 5 3
4 6 0 8 3
5 5 8 0 5
1 3 3 5 0

输出样例：

18

程序

import java.util.Scanner;
import java.util.Arrays;

public class Main
{
    public static final int N = 22;
    public static final int STATE = 1<<N;
    public static final int INF = 0x3f3f3f3f;
    public static void main(String[] args)
    {
        int[][] dis = new int[N][N];
        //f[i][j]表示当前遍历的所有点的状态为i,当前在点j的的最短距离
        int[][] f = new int[STATE][N];
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        for(int i = 0; i < n; i ++)
        {
            for(int j = 0; j < n; j ++)
            {
                dis[i][j] = sc.nextInt();
            }
        }
        //初始化
        for(int i = 0; i < STATE; i ++)
        {
            Arrays.fill(f[i], INF);
        }
        f[1][0] = 0;
        for(int i = 0; i < 1<<n; i ++)
        {
            for(int j = 0; j < n; j ++)
            {
                //有剪枝，所以时间复杂度可以宽松
                if(((i>>j)&1)!=0) //第j位为1,合法
                {
                    //k表示上一个位置
                    for(int k = 0; k < n; k ++)
                    {
                        if(((i>>k)&1)!=0)
                        {
                            //tip: 状态i-(1<<j)一定在状态i之前就算过了
                            f[i][j] = Math.min(f[i][j], f[i-(1<<j)][k]+dis[k][j]);
                        }
                    }
                }
            }
        }
        System.out.println(f[(1<<n)-1][n-1]);
    }
}
