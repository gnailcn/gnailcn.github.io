AcWing 41. 包含min函数的栈

2024-01-22 栈 思维 进阶指南 AcWing 算法

原题

设计一个支持push，pop，top等操作并且可以在O(1)时间内检索出最小元素的堆栈。

  • push(x)–将元素x插入栈中
  • pop()–移除栈顶元素
  • top()–得到栈顶元素
  • getMin()–得到栈中最小元素

数据范围

操作命令总数[0,100]。

样例

MinStack minStack = new MinStack();
minStack.push(-1);
minStack.push(3);
minStack.push(-4);
minStack.getMin();   --> Returns -4.
minStack.pop();
minStack.top();      --> Returns 3.
minStack.getMin();   --> Returns -1.

程序

class MinStack {
    Stack<Integer> dStack; //当前元素与下面的栈的最小值之间的差值
    int minv; //当前整个栈的最小值

    /** initialize your data structure here. */
    public MinStack() {
        dStack = new Stack<Integer>();
        minv = 0;
    }

    public void push(int x) {
        if(dStack.isEmpty())
        {
            dStack.push(-x);
            minv = x;
        }
        else
        {
            dStack.push(x-minv);
            if(x < minv)
            {
                minv = x;
            }
        }
    }

    public void pop() {
        if(dStack.peek()<0)
        {
            //当前栈顶元素就是最小值minv, 所以oldMinv-downMinV = peek()
            int d = dStack.peek();
            dStack.pop();
            minv = minv - d;
        }
        else
        {
            dStack.pop();
        }
    }

    public int top() {
        if(dStack.peek()<0)
        {
            //当前栈顶元素就是最小值minv
            return minv;
        }
        else
        {
            //minv刚好也是下面栈的最小值
            return dStack.peek() + minv;
        }
    }

    public int getMin() {
        return minv;
    }
}
