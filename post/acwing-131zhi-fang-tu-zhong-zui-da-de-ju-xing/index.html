AcWing 131.直方图中最大的矩形

2024-02-19 单调栈 算法提高课 思维 进阶指南 AcWing

原题

直方图是由在公共基线处对齐的一系列矩形组成的多边形。

矩形具有相等的宽度，但可以具有不同的高度。

例如，图例左侧显示了由高度为 2,1,4,5,1,3,3 的矩形组成的直方图，矩形的宽度都为 1：

通常，直方图用于表示离散分布，例如，文本中字符的频率。

现在，请你计算在公共基线处对齐的直方图中最大矩形的面积。

图例右图显示了所描绘直方图的最大对齐矩形。

输入格式

输入包含几个测试用例。

每个测试用例占据一行，用以描述一个直方图，并以整数 n 开始，表示组成直方图的矩形数目。

然后跟随 n 个整数 h1，…，hn。

这些数字以从左到右的顺序表示直方图的各个矩形的高度。

每个矩形的宽度为 1。

同行数字用空格隔开。

当输入用例为 n=0 时，结束输入，且该用例不用考虑。

输出格式

对于每一个测试用例，输出一个整数，代表指定直方图中最大矩形的区域面积。

每个数据占一行。

请注意，此矩形必须在公共基线处对齐。

数据范围

1≤n≤100000, 0≤hi≤1000000000

输入样例：

7 2 1 4 5 1 3 3
4 1000 1000 1000 1000
0

输出样例：

8
4000

程序：

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.BufferedWriter;
import java.io.OutputStreamWriter;
import java.util.Stack;

public class Main
{
    static final int N = (int)1e5 + 10;
    //注意下标从1开始
    // 每个元素左边第一个比它严格小的元素的idx
    static int[] left = new int[N];
    // 每个元素右边第一个比它严格小的元素的idx
    static int[] right = new int[N];
    static int[] nums = new int[N];
    public static void main(String[] args) throws IOException
    {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        while(true)
        {
            String[] input = br.readLine().split(" ");
            int n = Integer.parseInt(input[0]);
            if(n == 0)
            {
                break;
            }
            for(int i = 1; i <= n; i ++)
            {
                nums[i] = Integer.parseInt(input[i]);
            }
            //计算left数组
            //存储下标，并且从栈底到栈顶的元素对应的值是严格递增的
            Stack<Integer> s1 = new Stack<>();
            for(int i = 1; i <= n; i ++)
            {
                while(!s1.isEmpty() && nums[s1.peek()] >= nums[i])
                {
                    s1.pop();
                }
                if(s1.isEmpty())
                {
                    left[i] = 0;
                }
                else //nums[s1.peek()] < nums[i]
                {
                    left[i] = s1.peek();
                }
                s1.push(i);
            }
            //计算right数组
            //存储下标，并且从栈底到栈顶的元素对应的值是严格递增的
            Stack<Integer> s2 = new Stack<>();
            for(int i = n; i >= 1; i --)
            {
                while(!s2.isEmpty() && nums[s2.peek()] >= nums[i])
                {
                    s2.pop();
                }
                if(s2.isEmpty())
                {
                    right[i] = n + 1;
                }
                else //nums[s2.peek()] < nums[i]
                {
                    right[i] = s2.peek();
                }
                s2.push(i);
            }
            //计算答案
            long ans = 0;
            for(int i = 1; i <= n; i ++)
            {
                ans = Math.max(ans, (long)nums[i] * (right[i] - left[i] - 1));
            }
            bw.write(ans + "\n");
        }
        bw.flush();
    }
}
