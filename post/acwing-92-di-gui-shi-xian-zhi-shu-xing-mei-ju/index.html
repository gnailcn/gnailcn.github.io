AcWing 92. 递归实现指数型枚举

2023-10-15 进阶指南 AcWing 递归递推 算法

从 1∼n1 \sim n1∼n 这 nnn 个整数中随机选取任意多个，输出所有可能的选择方案。

输入格式

输入一个整数 nnn。

输出格式

每行输出一种方案。

同一行内的数必须升序排列，相邻两个数用恰好 111 个空格隔开。

对于没有选任何数的方案，输出空行。

本题有自定义校验器（SPJ），各行（不同方案）之间的顺序任意。

数据范围

1≤n≤151 \le n \le 151≤n≤15

输入样例：

3

输出样例：


3
2
2 3
1
1 3
1 2
1 2 3

程序

import java.util.Scanner;

public class Main
{
    static final int N = 16;
    static final boolean[] state = new boolean[N];
    static int n = 0;
    //当前正在选数t
    public static void dfs(int t)
    {
        if(t>n)
        {
            for(int i = 1; i <= n; i ++)
            {
                if(state[i])
                {
                    System.out.print(i);
                    System.out.print(" ");
                }
            }
            System.out.println();
            return;
        }
        //选择数t
        state[t] = true;
        dfs(t+1);
        //不选数t
        state[t] = false;
        dfs(t+1);
    }
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        n = sc.nextInt();
        dfs(1);
    }
}
