AtCoder ABC346.D Gomamayo Sequence

2024-03-24 AtCoder 思维 前缀和 算法

原题

Problem Statement

You are given a string SSS of length NNN consisting of 0 and 1.

A string TTT of length NNN consisting of 0 and 1 is a good string if and only if it satisfies the following condition:

  • There is exactly one integer iii such that 1≤i≤N−11 \leq i \leq N - 11≤i≤N−1 and the iii-th and (i+1)(i + 1)(i+1)-th characters of TTT are the same.

For each i=1,2,…,Ni = 1,2,\ldots, Ni=1,2,…,N, you can choose whether or not to perform the following operation once:

  • If the iii-th character of SSS is 0, replace it with 1, and vice versa. The cost of this operation, if performed, is CiC_iCi​.

Find the minimum total cost required to make SSS a good string.

Constraints

  • 2≤N≤2×1052 \leq N \leq 2 \times 10^52≤N≤2×105
  • SSS is a string of length NNN consisting of 0 and 1.
  • 1≤Ci≤1091 \leq C_i \leq 10^91≤Ci​≤109
  • NNN and CiC_iCi​ are integers.
