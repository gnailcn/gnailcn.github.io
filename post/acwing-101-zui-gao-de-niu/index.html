AcWing 101. 最高的牛

2023-10-28 进阶指南 AcWing 差分 算法

有 NNN 头牛站成一行，被编队为 1、2、3…N1、2、3…N1、2、3…N，每头牛的身高都为整数。

当且仅当两头牛中间的牛身高都比它们矮时，两头牛方可看到对方。

现在，我们只知道其中最高的牛是第 PPP 头，它的身高是 HHH ，剩余牛的身高未知。

但是，我们还知道这群牛之中存在着 MMM 对关系，每对关系都指明了某两头牛 AAA 和 BBB 可以相互看见。

求每头牛的身高的最大可能值是多少。

输入格式

第一行输入整数 N,P,H,MN, P, H, MN,P,H,M，数据用空格隔开。

接下来 MMM 行，每行输出两个整数 AAA 和 BBB ，代表牛 AAA 和牛 BBB 可以相互看见，数据用空格隔开。

输出格式

一共输出 NNN 行数据，每行输出一个整数。

第 iii 行输出的整数代表第 iii 头牛可能的最大身高。

数据范围

1≤N≤50001 \le N \le 50001≤N≤5000,
1≤H≤10000001 \le H \le 10000001≤H≤1000000,
1≤A,B≤100001 \le A,B \le 100001≤A,B≤10000,
0≤M≤100000 \le M \le 100000≤M≤10000

输入样例：

9 3 5 5
1 3
5 3
4 3
3 7
9 8

输出样例：

5
4
5
3
4
4
5
5
5

注意：

  • 此题中给出的关系对可能存在重复

程序

import java.util.Scanner;
import java.util.Objects;
import java.util.Set;
import java.util.HashSet;

public class Main
{
    public static class Pair
    {
        int a;
        int b;
        public Pair(int a, int b)
        {
            if(a <= b)
            {
                this.a = a;
                this.b = b;
            }
            else
            {
                this.a = b;
                this.b = a;
            }
        }
        public boolean equals(Object obj)
        {
            //同一个对象（地址相同）
            if(this == obj)
            {
                return true;
            }
            //obj是空对象，或者运行时类不同
            if(obj==null || getClass()!=obj.getClass())
            {
                return false;    
            }
            Pair myObj = (Pair)obj;
            return a==myObj.a && b==myObj.b;
        }
        public int hashCode()
        {
            return Objects.hash(a, b);
        }
    }
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        Set<Pair> S = new HashSet<Pair>();
        int n = sc.nextInt();
        int p = sc.nextInt();
        int h = sc.nextInt();
        int m = sc.nextInt();
        int[] height = new int[n+1]; //差分数组
        height[1] = h;
        for(int i = 1; i <= m; i ++)
        {
            int inputA = sc.nextInt();
            int inputB = sc.nextInt();
            Pair pair = new Pair(inputA, inputB);
            if(!S.contains(pair))
            {
                S.add(pair);
                //注意这里不要用inputA和inputB, 因为Pair内部改顺序了
                height[pair.a+1] -= 1;
                height[pair.b] += 1;
            }
        }
        for(int i = 1; i <= n; i ++)
        {
            height[i] += height[i-1];
            System.out.println(height[i]);
        }
    }
}
