AtCoder ABC333.C Repunit Trio

2024-03-13 AtCoder 模拟 算法

原题

Problem Statement

A repunit is an integer whose digits are all 111 in decimal representation. The repunits in ascending order are 1,11,111,…1, 11, 111, \ldots1,11,111,….

Find the NNN-th smallest integer that can be expressed as the sum of exactly three repunits.

Constraints

  • NNN is an integer between 111 and 333333333, inclusive.

Input

The input is given from Standard Input in the following format:

Copy

113

The integers that can be expressed as the sum of exactly three repunits are 3,13,23,33,113,…3, 13, 23, 33, 113, \ldots3,13,23,33,113,… in ascending order. For example, 113113113 can be expressed as 113=1+1+111113 = 1 + 1 + 111113=1+1+111.

Note that the three repunits do not have to be distinct.
