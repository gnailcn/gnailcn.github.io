Problem Statement

There is a grid with 10910^9109 by 10910^9109 squares. Let (i,j)(i, j)(i,j) denote the square at the (i+1)(i + 1)(i+1)-th row from the top and the (j+1)(j + 1)(j+1)-th column from the left (0≤i,j<109)(0 \leq i, j \lt 10^9)(0≤i,j<109). (Note the unusual index assignment.)
Each square is black or white. The color of the square (i,j)(i, j)(i,j) is represented by a character P[i mod N][j mod N]P[i \bmod N][j \bmod N]P[imodN][jmodN], where B means black, and W means white. Here, a mod ba \bmod bamodb denotes the remainder when aaa is divided by bbb.

Answer QQQ queries.
Each query gives you four integers A,B,C,DA, B, C, DA,B,C,D and asks you to find the number of black squares contained in the rectangular area with (A,B)(A, B)(A,B) as the top-left corner and (C,D)(C, D)(C,D) as the bottom-right corner.

Constraints

  • 1≤N≤10001 \leq N \leq 10001≤N≤1000
  • P[i][j]P[i][j]P[i][j] is W or B.
  • 1≤Q≤2×1051 \leq Q \leq 2 \times 10^51≤Q≤2×105
  • 0≤A≤C<1090 \leq A \leq C \lt 10^90≤A≤C<109
  • 0≤B≤D<1090 \leq B \leq D \lt 10^90≤B≤D<109
  • N,Q,A,B,C,DN, Q, A, B, C, DN,Q,A,B,C,D are all integers.
