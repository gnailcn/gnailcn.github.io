AcWing 788.逆序对的数量

2024-03-19 算法基础课 待完善 AcWing 排序 递归递推 算法

原题

给定一个长度为 nnn 的整数数列，请你计算数列中的逆序对的数量。

逆序对的定义如下：对于数列的第 iii 和第 jjj 个元素，如果满足 i<ji<ji<j 且 a[i]>a[j]a[i]>a[j]a[i]>a[j]，则其为一个逆序对；否则不是。

输入格式

第一行包含整数 nnn，表示数列的长度。

第二行包含 nnn 个整数，表示整个数列。

输出格式

输出一个整数，表示逆序对的个数。

数据范围

1≤n≤1000001\leq n\leq 1000001≤n≤100000，
数列中的元素的取值范围 [1,109][1,10^9][1,109]。

输入样例：

6
2 3 4 5 6 1

输出样例：

5

程序

import java.util.Scanner;

public class Main
{
    static int[] nums;

    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        nums = new int[n];
        for (int i = 0; i < n; i++)
        {
            nums[i] = sc.nextInt();
        }
        long ans = mergeSort(0, n - 1);
        System.out.println(ans);
    }

    //归并排序，并且返回逆序对的数量
    public static long mergeSort(int l, int r)
    {
        if(l >= r)
        {
            return 0;
        }
        int mid = (l + r) / 2;
        long lcnt = mergeSort(l, mid);
        long rcnt = mergeSort(mid + 1, r);
        long cnt = 0;
        int[] temp = new int[r - l + 1];
        int i = l, j = mid + 1, k = 0;
        while(i <= mid && j <= r)
        {
            if(nums[i] <= nums[j])
            {
                temp[k++] = nums[i++];
            }
            else //左边的数大于右边的数
            {
                temp[k++] = nums[j++];
                cnt += mid - i + 1; //逆序对的数量
            }
        }
        while(i <= mid)
        {
            temp[k++] = nums[i++];
        }
        while(j <= r)
        {
            temp[k++] = nums[j++];
        }
        for(i = l, k = 0; i <= r; i++, k++)
        {
            nums[i] = temp[k];
        }
        return lcnt + rcnt + cnt;
    }
}
