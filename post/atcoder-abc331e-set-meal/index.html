AtCoder ABC331.E Set Meal

2024-03-24 字典 AtCoder 模拟 算法

Problem Statement

AtCoder cafeteria sells meals consisting of a main dish and a side dish.
There are NNN types of main dishes, called main dish 111, main dish 222, …\dots…, main dish NNN. Main dish iii costs aia_iai​ yen.
There are MMM types of side dishes, called side dish 111, side dish 222, …\dots…, side dish MMM. Side dish iii costs bib_ibi​ yen.

A set meal is composed by choosing one main dish and one side dish. The price of a set meal is the sum of the prices of the chosen main dish and side dish.
However, for LLL distinct pairs (c1,d1),…,(cL,dL)(c_1, d_1), \dots, (c_L, d_L)(c1​,d1​),…,(cL​,dL​), the set meal consisting of main dish cic_ici​ and side dish did_idi​ is not offered because they do not go well together.
That is, NM−LNM - LNM−L set meals are offered. (The constraints guarantee that at least one set meal is offered.)

Find the price of the most expensive set meal offered.

  • 1≤N,M≤1051 \leq N, M \leq 10^51≤N,M≤105

  • 1≤ai,bi≤1091 \leq a_i, b_i \leq 10^91≤ai​,bi​≤109
  • 1≤ci≤N1 \leq c_i \leq N1≤ci​≤N
  • 1≤dj≤M1 \leq d_j \leq M1≤dj​≤M

  • All input values are integers.
